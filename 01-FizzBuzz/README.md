# 01-FizzBuzz

## Usage
```
$ java -jar 01-FizzBuzz-1.0.jar 4 7
```
## How to build
To build and execute checks use the following command
```
$ gradle check
```
To create an executable jar-file into the root directory of the project use the following command
```
$ gradle jar
```
### Expected output
```
4, Buzz, Fizz, 7
```
